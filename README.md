# Basic HTML Website

This project is my solution to the [Basic HTML Website](https://roadmap.sh/projects/basic-html-website) challenge on roadmap.sh.

## 🚀 Live Demo
> **[Click here to view the Project Page](https://tawuf-israk.github.io/basic-html-website/index.html)**

## 📄 Project Description
This project demonstrates the creation of a multi-page website using semantic HTML. It contains no CSS or JavaScript, focusing purely on structure and linking.

The website includes:
- **Homepage**: Introduction
- **Projects**: A list of work
- **Articles**: Some writing content
- **Contact**: Basic contact information
